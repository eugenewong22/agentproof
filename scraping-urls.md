## 3. Oxylabs Scraping Notes

### SSOC (SingStat)
- **Best targets:** The direct XLSX/XLS download links above — these contain the full structured SSOC code data (hierarchical classification, definitions, correspondence tables).
- **The SSOC Search page** (`/ssoc-search`) is a JS-rendered search form with no visible API endpoint in static HTML. The actual data lives in the XLSX files above. If you need to scrape the search tool, you'll need to inspect network requests in browser devtools to find the underlying API.
- **File format:** All files are static at `/files/{uuid}.{ext}` on `singstat.gov.sg` — no auth required.

### SkillsFuture Frameworks
- Each sector page (e.g. `/skills-framework/ict`) contains career pathways, job roles, skills/competencies (TSC and CCS), and training programme mappings.
- Pages are JS-rendered (React/Next.js) — the skill/competency data is likely loaded via API calls. Inspect network tab to find the XHR/fetch endpoints for each sector's data.
- Some sectors may also have downloadable PDF guides linked from their pages.

### Recommended Approach
1. **SSOC codes:** Download the XLSX files directly (no scraping needed) — start with SSOC 2024 Classification Structure + Detailed Definitions.
2. **SkillsFuture:** Use Oxylabs to scrape each sector page listed above. You may also want to intercept the API calls each page makes to get structured JSON directly.
3. **Mapping:** Use the SSOC ↔ ISCO-08 correspondence tables to map between SSOC codes and international occupation standards.

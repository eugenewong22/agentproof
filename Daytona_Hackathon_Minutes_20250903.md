### 3. Sponsor Introductions (3:40 - 15:00)

#### 3.1 Daytona - Safe AI Sandbox (5:55 - 10:30)

**What it is:**
- Secure sandbox environment for AI agents
- Protects your laptop and data from accidental damage
- Cloud-based virtual machines (not local)

**Key Features:**
- Sandboxes live in the cloud, not on your machine
- Can scale and expand as needed
- Typical sandbox lifespan: 90-200 minutes
- Supports multiple languages: TypeScript, Java, etc.

**MCP (Model Context Protocol):**
- Helps AI agents (Claude, Codex, DPC, etc.) connect to different devices safely
- Keeps code contained in sandbox
- Prevents unauthorized device access
- Critical for security: "If your agent runs PowerShell scripts with admin rights on your device - that would be very bad"

**Access Methods:**
- **UI Dashboard** - Visual interface
- **CLI** - Command line (PowerShell, Mac terminal)
- **Python SDK** - For programmatic access

**Demo Highlights:**
- Sandbox creation takes seconds
- Terminal-based interface for code execution
- Works similar to Python scripts
- AI assistants available within sandbox to help

#### 3.3 NOSANA - Decentralized GPU Computing (10:46 - 12:35)

**What it is:**
- Decentralized GPU network
- Alternative to AWS/Google Cloud for GPU access
- 6x cheaper than traditional cloud providers

**Key Features:**
- GPU nodes distributed worldwide
- For AI model training and inference
- Prevents laptop overheating during heavy computation
- Multiple GPU types available
- Choose location closest to you for better performance

**Pricing:**
- Per-minute billing
- Significantly cheaper than AWS
- Credits provided for hackathon use
- Can deploy to different global locations

**Deployment Process:**
1. Go to NOSANA.com
2. Create API key (required for all services)
3. Navigate to deployment section
4. Select GPU node type (e.g., 8GB VRAM)
5. Choose location based on proximity
6. Review scores and pricing
7. Deploy

**Important Notes:**
- Different nodes have different performance scores
- Don't all choose the same node - it will get congested
- Check node scores before deploying
- Some nodes may not be immediately available - wait or try another

**Action Items:**
- [ ] Create NOSANA account and API key
- [ ] Review available GPU nodes
- [ ] Select appropriate node for your needs
